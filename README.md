# Super Smart Home - Capstone Project

## Topic
A business startup called Aurora that offers a product called SASH (Super Artificial Samrt Home) that is fully automated, privacy‑focused smart home system. Everything runs on a local server – no cloud, no subscriptions. I'm interested in growing this idea into a future start-up

## Team Member Roles
I am the only team member. 

## Design Decisions

- Purple accents for calls‑to‑action and highlights.
- All pages use a shared navigation bar for consistency.
